# Sabatora0843's Homebrew Tap

Custom [Homebrew](https://brew.sh/) formulae for my CLI tools and utilities.

![Homebrew](https://img.shields.io/badge/Homebrew-Tap-orange)
![License](https://img.shields.io/badge/License-MIT-blue)

This tap provides Homebrew formulae for tools developed and maintained by **Sabatora0843**.

## Installation

First, add this tap:

```bash
brew tap sabatora0843/tap
```

Then install a formula:

```bash
brew install <formula>
```

Alternatively, install directly from the tap in one command:

```bash
brew install sabatora0843/tap/<formula>
```

## Available Formulae

| Formula | Description |
|--------|-------------|
| keybox | A lightweight CLI tool for managing secrets using the macOS Keychain. |

## Uninstall

To uninstall a specific formula:

```bash
brew uninstall <formula>
```

If you no longer need this tap, remove it with:

```bash
brew untap sabatora0843/tap
```

## About Homebrew Tap

A Homebrew tap is a repository containing custom formulae that can be installed using the `brew` command.

For more information, see the official Homebrew documentation:

https://docs.brew.sh/Taps
